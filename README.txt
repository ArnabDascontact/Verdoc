VerDoc bundle
================

Files:
- index.html — main website
- app.js — front-end logic
- doctor_data.js — dataset for the browser
- doctor_data.json — JSON version of the dataset
- favicon.png — browser icon
- logo-mark.png — header logo
- logo-theme.png — large background logo
- VerDoc.png — alternate branding image

Preview rules in this build:
- All verified records are locked.
- Free preview shows up to 100 non-verified leads per country.
- Premium access is handled by request.

Open index.html in a browser to view the site.

Notes:
- The site now uses a Terms and Conditions section in the footer.
- Free preview copy actions only copy the email and do not open the request-access modal.
- Have counsel review the published terms before launch.
